AI Mock Interview Platform

Overview

AI Mock Interview Platform helps candidates prepare for interviews by analyzing their resume and job description, generating relevant interview questions, and providing a readiness assessment.

Features

- Resume Upload (PDF)
- Job Description Input
- AI-Based Interview Question Generation
- Interview Analysis
- Readiness Score
- Hiring Recommendation

Tech Stack

- Python
- Flask
- HTML
- CSS
- Google Gemini API

How It Works

1. Upload your resume.
2. Paste the target job description.
3. Generate interview questions based on the candidate profile.
4. Review interview analysis and readiness score.

Future Enhancements

- Real-time AI answer evaluation
- Voice-based mock interviews
- Skill gap analysis
- Personalized learning recommendations
